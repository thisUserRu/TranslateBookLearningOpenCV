### Примечания автора:

(!) - пометка переводчиком некачественного перевода

{} - пояснения переводчика

H2 - размер заголовка главы (##)

H3 - размер подзаголовка главы (###)

**Процессы:**
```
[] - процесс завершен
(] - процесс не завершен
() - процесс не начинался
П - перевод
РС - собственная редактура
РП - профессиональная редактура
```
